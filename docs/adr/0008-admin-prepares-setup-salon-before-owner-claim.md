# Admin prepares a Setup Salon before owner claim

Authorized platform staff may create a non-public Setup Salon and prepare its initial operating data before a user owns it. A one-time Salon Handoff verifies the intended owner's phone, establishes their login access, activates the salon, and ends ordinary platform setup access. We chose this over requiring owner signup first or introducing a separate setup-case workflow because a small field-sales team needs the shortest path from an in-person agreement to a usable salon while ownership still remains tied to the owner's verified identity.
