آراویرا — راهنمای کوتاه مدیر

برنامهٔ نوبت‌دهی سالن. با شماره موبایل + رمز وارد شوید؛ بدون اینترنت کار نمی‌کند.

نوار پایین (مدیر)

تقویممشتریانداشبوردتنظیمات
نوبت‌هالیست مشتریآمار کلیساعت، خدمات، پرسنل، خروج

پرسنل: فقط تقویم + تنظیمات؛ نمی‌تواند نوبت بسازد یا ویرایش/حذف کند — فقط می‌بیند.

تقویم

  • روز / هفته / ماه / لیست — تاریخ شمسی.
  • نوبت جدید: دکمهٔ + (پایین چپ) یا زدن روی زمان خالی.
  • فیلتر: نام پرسنل را بزنید تا فقط نوبت‌های او دیده شود.

روی نوبت بزنید: جزئیات، تماس با مشتری، دکمه‌های وضعیت، ویرایش، حذف.

وضعیتمعنی کوتاه
در انتظارثبت شده، قطعی نیست → تایید یا لغو
تایید شدهقطعی است → انجام شد یا غیبت
انجام شده / لغو شده / غیبتپایان مسیر

ویرایش کامل: «ویرایش نوبت» → ذخیره. حذف: آیکن سطل → تأیید.

مشتریان

جستجو (نام/شماره) · مشتری جدید · ویرایش از لیست. برای نوبت از همین لیست انتخاب می‌شود.

داشبورد

امروز / هفته / ماه · درآمد ماه (فقط انجام‌شده) · مشتریان و پرسنل · پرطرفدارترین خدمت · بار کاری پرسنل. خودش هر ~۱ دقیقه به‌روز می‌شود.

تنظیمات

  • ساعات کاری + فاصلهٔ اسلات (دقیقه) → حتماً ذخیره.
  • خدمات: جدید / ویرایش؛ غیرفعال = موقتاً برای نوبت جدید مناسب نیست.
  • حالت تاریک — سلیقه‌ای.
  • پرسنل جدید: نام، موبایل، رمز، نقش (مدیر/پرسنل).
  • خروج از حساب — پایین صفحه.

پرسنل و اعلان

اعلان نوبت (اگر در تنظیمات پرسنل دیده شد): با روشن بودنش + نصب روی صفحهٔ اصلی، وقتی مدیر برای همان پرسنل نوبت بگذارد اعلان می‌آید. نیاز به تنظیم فنی سرور دارد.

هر روز ۳ کار

  1. تقویم امروز را چک کن — وضعیت نوبت‌ها را درست کن.
  2. نوبت جدید با + یا زمان خالی.
  3. عوض شدن ساعت کاری/خدمات → تنظیمات.
