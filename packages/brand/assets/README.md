# Saluna brand masters

These PNGs are the canonical raster sources for generated PWA and web assets.

- `saluna-mark-gradient.png`: detailed mark for product UI and large brand use.
- `saluna-mark-flat.png`: simplified mark for favicons and install icons.
- `saluna-logo-horizontal-fa.png`: Persian horizontal wordmark.
- `saluna-logo-horizontal-fa-en.png`: Persian/English horizontal wordmark.
- `saluna-logo-stacked-fa-en.png`: stacked lockup used on launch screens.

Run `pnpm brand:icons` after changing a master.
