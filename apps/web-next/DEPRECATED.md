# Deprecated — `@repo/web-next`

This package is the **legacy Next.js** public marketing + booking site. It is kept for reference and rollback only.

**Use `@repo/web` (Astro) instead** — the current public site in `apps/web`.

```bash
pnpm dev:web          # Astro public site (port 3001)
pnpm dev:web-next     # this legacy Next app (port 3001 — do not run both at once)
```

Same pattern as `apps/app` (legacy manager PWA) vs `apps/pwa`.
